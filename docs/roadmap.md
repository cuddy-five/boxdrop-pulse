# Roadmap

## Phase 1 - MVP
- QBO webhook receiver + queue
- Worker flow to fetch QBO data
- SMS rating request + inbound parsing
- Owner alert for low scores
- Basic Postgres logs and dedupe

## Phase 2 - UX polish
- Review request UX and branded copy
- Staff follow-up notes
- Dashboard for session search

## Phase 3 - Optimization
- Automatic retry policies
- Alerts and metrics
- Deeper QBO updates (custom fields)
